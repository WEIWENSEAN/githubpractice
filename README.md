# GitHub Practice

透過這個Repository練習Git及GitHub

## 分支

- master: 主要分支
- develop: 開發分支
- release: 發布分支
- feature: 功能分支
- issue: 問題分支
- hotfix: 修補分支

## GitHub Flow

https://guides.github.com/introduction/flow/

## Fork

將Repository複製至自己的GitHub帳號下，此機制為GitHub的權限控管，避免影響主要程式

## Pull Request

透過Pull Request(PR)送出請求，請求原作者將異動合併至原Repository
